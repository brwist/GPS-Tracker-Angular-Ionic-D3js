WiFi Tracking (Default)

While motion is detected, Trakkit checks for WiFi every 2 minutes.
If an open WiFi is found, Trakkit sends its WiFi location (blue dots).
This is an estimated location based on the location of the WiFi router. GPS is not used.
Typical battery life is 2 months.

GPS Tracking

Same as WiFi Tracking mode, except the GPS is used for precise location.
If an open WiFi is found, Trakkit records and shows its GPS position (red dots).
If Trakkit cannot get a GPS position (it is cloudy, rainy, etc) then it resolves on an estimated position based on the location of the WiFi source (blue dots).
Typical battery life less than a month.

GPS Logging

When Trakkit senses no motion, it checks for WiFi and if it finds one it will try to check in (you'll see a red dot if position is GPS based or blue dot if position is WiFi based).
If you are near a Home WiFi, previously configured, Trakkit uploads all its GPS Logging data (black dots).
Logging Data includes: Time, Location, Speed, Heading, Motion Level, and Temperature
Typical battery life is 2 months.

Sharing

This feature allows you to Share your Trakkit information with a friend if they have the Trakkit App installed.
Once you share your Trakkit with them, they will be able to set Rules and Trigger Alerts, but will not be able to change or Edit any of your settings.
